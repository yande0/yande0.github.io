Ubuntu新手推荐配置

发表于 2018-02-24 | 分类于 推荐

​ 以前因为win7老是卡欢迎，所以换了Ubuntu，用了之后觉得还不错。不过尽管Ubuntu是Linux中对新手友好的系统，但新手还是要多花点功夫才能适应。这次我来讲讲Ubuntu新手怎么做让新装的Ubuntu更好操作一点。
​ 假如你是笔记本电脑连WiFi的用户，你肯定会发现一个问题——电脑连不上WiFi。这是因为笔记本上的无线网卡驱动不是开源的，Ubuntu不能把它集成到自家系统里。那么怎么样迅速解决这个问题呢？我们需要连一个有线且不需要设置宽带的网线接进去（无线路由器的LAN口、办公室网线、手机USB共享网络等等）。然后进入系统设置-软件和更新，点进附加选项，可以看到有802无线网卡驱动，用它的专有驱动就行了。假如刷新不出来驱动的话，先在Ubuntu软件里设置一下服务器，用清华大学的或者163或者阿里云的都行，假如你住的地方偏远的话可以让系统自己测试一下哪个镜像站速度最快，选完之后让系统自己更新一下。之后肯定能见到802无线网卡的驱动。这样就解决了Ubuntu不能用WiFi的问题。

​ 再往下就该是装软件了，终于到这一步了啊。前期装软件apt命令绝对是最常用的。我个人先推荐安装几个软件：

  • 新立得包管理器

非常厉害的一个软件，Ubuntu必装。管理你下载的安装包。安装命令

  • 文泉驿字体

文泉驿字体是开源的字体，挺好看的，很适合作为系统默认字体。安装命令

  • Unity Tweak Tool

这个是用来设置桌面的，绝对比自带的强大的得多。可以在这里面设置文泉驿微米黑为默认字体。安装命令

  • GDebi

这个是用来安装下面要推荐的软件的deb包的。尽管Ubuntu自带的软件市场也能做到，但是软件市场安装deb包不太稳定，有时候玩命安装也安装不上，和Ubuntu早期使用的这个软件相比差得很多。安装命令

  • WPS

我相信WPS你使用起来比Ubuntu自带的LibreOffice得心应手得多。这个就没安装命令了，因为是闭源的。用deb包安装。下载地址http://community.wps.cn/download/ 选对应的版本下载。下载了之后右键用GDebi打开。顺带，WPS打开之后会提示系统字体缺失，无视即可，缺失的字体假如不是专业人员的话是很难去使用的。Word打开之后首页默认是Docer，一堆广告，实际上那个页面右下角有设置，可以设置它不自动弹出。

  • Crossover

Wine的商业版，可以运行很多Windows上的软件，功能非常强悍。最推崇用它安装TM2013，因为QQ没有适配Linux，而用wine配置QQ会出现各种各样的问题，而办公特化的QQ——TM2013则是可以很轻易的用wine安装。下载地址https://www.codeweavers.com/products/crossover-linux，支持正版。

  • Chrome浏览器

尽管自带的Firefox已经非常好用了，但我还是推荐安装Chrome浏览器，这是因为Chrome有一项黑科技——ARChon，这项黑科技可以让你的Chrome浏览器运行安卓APP。黑科技我们以后再说，先安装Chrome。现在只有64位。https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb

  • App Grid

App Grid 是一款轻量级的Ubuntu软件中心，可安装闭源软件。比自带软件中心强，体积还小得多。安装命令有三条，按顺序运行
