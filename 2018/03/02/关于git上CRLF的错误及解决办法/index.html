关于git上CRLF的错误及解决办法

发表于 2018-03-02

相信用过git的Windows用户都见过这个warning

看着不舒服，一出现一大溜。不过用Linux的git的应该没见过，区别就在此。

原因呢，就是Linux和Windows下的换行符不一样，Linux下是LF，Windows下是CRLF。用Windows保存文本的时候换行符自动转为CRLF，才会出现这种情况。

解决方法也就一行代码的事儿，禁止git自动转换换行符
